Get Started in 5 Minutes

Follow these simple steps to accept your first payment.

1

Install the SDK

Install the PayFlow Node.js library using npm or yarn.

terminal
# Using npm
npm install payflow-sdk

# Using yarn
yarn add payflow-sdk
2

Initialize the client

Set up the PayFlow client with your API keys.

app.js
const payflow = require('payflow-sdk');

// Initialize with your secret key
const client = payflow('sk_test_your_secret_key');
3

Create a payment

Process a payment with just a few lines of code.

app.js
// Create a payment
async function createPayment() {
  try {
    const payment = await client.payments.create({
      amount: 2000, // $20.00
      currency: 'usd',
      source: 'tok_visa', // Token from frontend
      description: 'Payment for order #12345'
    });
    console.log('Payment successful:', payment.id);
  } catch (error) {
    console.error('Payment failed:', error);
  }
}
4

Handle the response

Process the payment response and update your application state.

app.js
// Check payment status
if (payment.status === 'succeeded') {
  // Update order status in your database
  await updateOrderStatus(orderId, 'paid');
  sendConfirmationEmail(customerEmail);
} else {
  // Handle failed payment
  handlePaymentFailure(payment);
}

API Reference

Explore our REST API with detailed documentation and examples.

Create a Payment

POST /v1/payments

Creates a payment to charge a credit card or other payment source.

Request

shell
curl https://api.payflow.com/v1/payments \
  -u sk_test_your_secret_key: \
  -d amount=2000 \
  -d currency=usd \
  -d source=tok_visa \
  -d description="Payment for order #12345"

Response

json
{
  "id": "pay_1Jk2Gd2eZvKYlo2CxzCx1234",
  "object": "payment",
  "amount": 2000,
  "currency": "usd",
  "status": "succeeded",
  "description": "Payment for order #12345",
  "created": 1629999999
}
