Smart Payment Routing

Our intelligent routing system automatically selects the best payment processor for each transaction based on cost, success rates, and regional preferences.

  • Increase authorization rates by up to 5%
  • Reduce processing costs with optimal routing
  • Custom rules for specific business needs
Learn about payment routing
routing.js
// Configure smart routing rules
const routingConfig = {
  strategy: 'optimize_for_success_rate',
  fallback: 'lowest_cost',
  rules: [
    {
      condition: 'amount > 1000',
      processor: 'premium_gateway'
    },
    {
      condition: 'currency === "EUR"',
      processor: 'european_provider'
    }
  ]
};

// Apply routing to payment
const payment = await payflow.payments.create({
  amount: 2500,
  currency: 'usd',
  routing: routingConfig
});
